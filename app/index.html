Help for SHRDLURN (esc to close)

The goal is to teach SHRDLURN (the computer) to understand more English. You have to define things in a way that can already be understood by SHRDLURN. This includes anything that you or others have already taught it, as well as the core language below, which it always understands:

  • number: 0, 1, 2, ..., 9, many
  • color: red, orange, yellow, green, blue, white, black, pink, and brown.
  • action: remove, add [color]
  • direction: top, bot, left, right, front, back, very left
  • control: [action] ; [action], repeat 3 [ action ]
    • repeat 3 [ remove red ]
    • repeat 2 [ add red all ; add pink ]
  • sets: select
    • select very top of all
    • select has color red
    • select this and not very left and not very right
    • select nothing
    • for has color brown [ add green ]
    • for very top of has color red [ repeat 2 [ add yellow left ] ]
    • isolate selected [ add red left ]
  • move: move [direction] (this moves the selected cubes)
      If you first select blocks, you can then move them like so:
    • move left
    • repeat 3 [ move left ]
  • update: (updates/changes current selection)
    • update color green
